# Painel Logística 2026 (GitHub Pages)

Versão simplificada pra rodar estático no GitHub Pages.

## Rodar local
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Dados
Troque os JSONs em `public/data/`.

## Custos (guard)
Senha padrão: `MACPONTA2026` (troca em `src/hooks/useCustosAuth.jsx`).

> Obs: isso não é segurança real. É só pra não deixar “fácil”.
